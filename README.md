# Elite Tech Tasks

A React productivity application — with a neon‑dark Elite Tech vibe.  
This project showcases:
- **Task Board** → create, edit, and delete tasks
- **Categories & Status** → organize tasks by progress
- **Persistent Storage** → tasks saved locally
- **Responsive UI** → styled with cyberpunk‑inspired CSS
